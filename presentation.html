WHY

왜 "자기주도적 학습" 인가

  • 운전 지식은 한 번 익히고 끝나는 게 아니라 평생 활용되는 안전 지식
  • 학습자 다양성 확대 — 외국인 운전자, 고령자, 면허 재취득자 → 획일적 교재로는 한계
  • 학습자가 자기 약점·속도·범위를 직접 통제할 수 있어야 학습이 지속됨
학습자 다양성 → 다국어 지원
FRAMEWORK

Zimmerman 자기조절학습(SRL) 3단계

계획 · Forethought

무엇을, 어떻게 학습할지 학습자가 직접 결정

수행 · Performance

개념 → 가이드 → 연습 → 실전을 한 흐름으로 학습

성찰 · Self-reflection

본인의 학습 곡선을 점검하고 다음 단계로 교정

계획 → 수행 → 성찰 ↺
→ 우리 앱은 이 3단계를 한 앱 안에서 닫힌 사이클로 제공
STAGE 1 · FORETHOUGHT

계획 — 학습자가 범위를 고른다

  • 학습하기 — 10개 소카테고리(주류 / 어린이보호구역 / 긴급상황 등) 중 약한 영역 선택
  • 문제 풀기 — 4가지 모드(전체 / 미풀이 / 즐겨찾기 / 오답) 학습자가 직접 결정
  • 즐겨찾기 — 본인 판단으로 중요한 항목을 직접 큐레이션
